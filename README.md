# Privacy Policy – ​​Seriport

**Last Updated:** 2026-05-22

## 1. Overview

Seriport is a desktop application that allows users to communicate with serial port (COM port) devices. This privacy policy explains how the application handles your personal data.

## 2. Data Collected

Seriport **does not collect, process, or share any personal data with third parties.**

The application only stores the following data **locally** on the user's device:

- Serial port settings (baud rate, data bit, parity bit, etc.)
- Application preferences and theme settings
- Data history sent/received (only on the device, upon user request)

This data is never transmitted over the internet or sent to our servers.

## 3. Internet Access

Seriport does not **use an internet connection.** All operations are performed entirely locally (offline).

## 4. Third-Party Services

The application **does not use** any third-party analytics, advertising, or tracking services.

## 5. Data Security

All user settings and preferences are stored only on the local device at the following location:

```
%APPDATA%\Seriport\appsettings.json
```

Only the user can access this file.

## 6. Children's Privacy

The application does not knowingly collect data from children under the age of 13.

## 7. Changes

This privacy policy may be updated from time to time. Changes will be announced on this page.

## 8. Contact

For questions regarding the privacy policy, please open an **Issue** on GitHub:

https://github.com/yukselacikgoz/seriport-privacy/tree/main

# Gizlilik Politikası – Seriport

**Son Güncelleme:** 22-05-2026

## 1. Genel Bakış

Seriport uygulaması, kullanıcıların seri port (COM port) cihazlarıyla iletişim kurmasını sağlayan bir masaüstü uygulamasıdır. Bu gizlilik politikası, uygulamanın kişisel verilerinizi nasıl ele aldığını açıklamaktadır.

## 2. Toplanan Veriler

Seriport **hiçbir kişisel veri toplamaz, işlemez veya üçüncü taraflarla paylaşmaz.**

Uygulama yalnızca aşağıdaki verileri **yerel olarak** kullanıcının cihazında saklar:

- Seri port ayarları (baud rate, veri biti, eşlik biti vb.)
- Uygulama tercihleri ve tema ayarları
- Gönderilen/alınan veri geçmişi (yalnızca cihazda, kullanıcı isteğiyle)

Bu veriler hiçbir şekilde internet üzerinden iletilmez veya sunucularımıza gönderilmez.

## 3. İnternet Erişimi

Seriport uygulaması **internet bağlantısı kullanmaz.** Tüm işlemler tamamen yerel olarak (offline) gerçekleştirilir.

## 4. Üçüncü Taraf Hizmetler

Uygulama herhangi bir üçüncü taraf analitik, reklam veya izleme hizmeti **kullanmamaktadır.**

## 5. Veri Güvenliği

Kullanıcıya ait tüm ayarlar ve tercihler yalnızca yerel cihazda şu konumda saklanır:

```
%APPDATA%\Seriport\appsettings.json
```

Bu dosyaya yalnızca kullanıcının kendisi erişebilir.

## 6. Çocukların Gizliliği

Uygulama, 13 yaşın altındaki çocuklardan bilerek veri toplamamaktadır.

## 7. Değişiklikler

Bu gizlilik politikası zaman zaman güncellenebilir. Değişiklikler bu sayfa üzerinden duyurulacaktır.
